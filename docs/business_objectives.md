# Business Objectives

## Project Overview
Machine Learning project for breast cancer detection using the Wisconsin Diagnostic Breast Cancer dataset.

## Main Objectives
1. Implement and compare 6+ ML algorithms for breast cancer detection
2. Reproduce and improve research paper results
3. Follow CRISP-DM methodology throughout the project

## Algorithms to Implement
- GRU-SVM
- Linear Regression
- MLP (Multi-Layer Perceptron)
- KNN (K-Nearest Neighbors)
- Softmax Regression
- SVM (Support Vector Machine)

## Success Metrics
- Model accuracy
- Precision and recall
- F1-score
- ROC-AUC score
